# Daily Update Template

```
[YYYY-MM-DD] Progress
- Completed: [task IDs and brief descriptions]
- In Progress: [task IDs and brief descriptions]
- Blockers: [any issues preventing progress]
- Tomorrow: [planned tasks for next day]
```

## Example

```
[2025-08-03] Progress
- Completed: AI-01 (environment setup), ENV-01 (Rust installation)
- In Progress: AI-02 (Asset Report)
- Blockers: Missing intent templates
- Tomorrow: Begin LC-01 after asset delivery
```
